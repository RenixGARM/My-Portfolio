# Roberto Chamorro - Portfolio

![Portfolio Preview](./public/global-bg.jpg)

Bienvenido al código fuente de mi portfolio profesional. Este proyecto es una Single Page Application (SPA) construida con tecnologías web modernas, diseñada para destacar mi experiencia en automatización, IA generativa y desarrollo frontend, sin escribir código a mano, utilizando metodologías de interacción "Vibe Coding" con IA.

## Tecnologías Principales

- **React 19**: Biblioteca UI para la construcción de componentes.
- **Vite**: Entorno de desarrollo ultrarrápido y empaquetador.
- **Tailwind CSS v4**: Framework de utilidades CSS para diseño responsivo y sistema de diseño adaptativo (Light/Dark mode).
- **GSAP (GreenSock)**: Animaciones de alto rendimiento, efectos magnéticos y transiciones de scroll suaves.
- **React Router Dom**: Enrutamiento del lado del cliente.
- **TS Particles**: Sistema ligero de partículas interactivas en el fondo.

## Funcionalidades Destacadas

- 🌗 **Dark/Light Mode Dinámico**: Implementación fluida que invierte colores manteniendo íntegros los acentos de la marca mediante variables CSS adaptativas sin depender del config de Tailwind.
- 🧲 **Botones Magnéticos**: Interacciones avanzadas con el cursor gestionadas mediante interpolación espacial y GSAP.
- ✨ **Partículas Interactivas**: Fondo reactivo al ratón, completamente optimizado para no penalizar el rendimiento.
- 🍪 **Banner de Cookies Legal**: Construido enteramente con estado local (Local Storage) y conforme a las normativas de privacidad (GDPR).
- 📱 **Diseño 100% Responsivo**: Todas las interfaces y elementos interactivos se adaptan desde pantallas ultra anchas hasta dispositivos móviles.

## Desarrollo Local

Si deseas ejecutar este proyecto en tu entorno local:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/RenixGARM/My-Portfolio.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
